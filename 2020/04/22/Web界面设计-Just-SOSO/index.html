Web界面设计 - Just SOSO

资源查询系统的设计与实现 - 需求分析

概述

本系统主要为用户提供一个快捷，方便的的查询学习资源，并且分享资源的大众平台。

项目功能结构图

本系统一般游客可以查询共享资料，只有经过注册、登录的合法用户才可以查询私有资源和发布资源和进行后台管理，其架构如图所示。

项目 说明
登录系统 包括注册、登录、修改密码、找回密码等
前台 一般用户或游客可直接浏览的页面
后台 系统管理员通过后台对系统的数据进行管理和维护

登录系统功能结构

为用户提供进入系统的正确合法方式，其功能架构如图所示。

功能 描述
注册 提供用户输入信息的界面，使得用户成为本网站的合法用户
登录 用户以合法身份进入本网站，以便发表新闻评论或这管理网站内容
找回密码 提供一个用户忘记密码后找回密码的途径
修改密码 用户可根据需要修改自己的密码

前台功能架构

用户进入网站后可以直接通过浏览主页，其中分有资源大分类表项以及资源细分表项。用户可以依照需求进行查询。注册用户可以发表资源至网站和评论资源。

功能 描述
主页 为用户提供一个便于查询的统一集合界面
资源分类列表 对不同资源进行类别划分
资源详细内容页 对资源内容的详细展示，为用户提供清晰的内容解读
资源发表 注册用户通过资源发表页，发布资源
评论资源 注册用户可以发表评论来答谢发布者

后台功能结构

功能 描述
后台主页 为后台管理提供一个统一的界面
用户管理 对角色、功能、权限等管理
资源管理 资源类别、发布、审核、删除不良资源等
实时统计 实时统计用户在一定时段间下载的资源
评论管理 审核，删除不良评论
用户管理
功能 描述
角色管理 不同的角色可以进入不同的功能界面
系统功能管理 增删改查系统功能页面（个人主页与系统界面）
权限管理 修改不同角色可操作的页面（如管理员与一般用户）
用户管理 查询，删除特定用户。使用监管制，禁止某些用户使用本系统
资源管理
功能 描述
资源类别管理 增删改查资源的不同种类
发布资源 上传资源后等待审核
审核资源 审核资源内容是否符合要求
删除资源 删除不符合要求的资源
实时统计
功能 描述
资源发布数统计 按时间段统计不同类别的资源
访问量统计 统计出访问量前10位的资源
下载量统计 按时间段统计出下载量前10位的资源
评论量统计 统计出评论量前10位的精品资源
评论管理
功能 描述
审核评论 审核用户发布的评论是否符合要求
删除评论 删除不符合要求的不良评论

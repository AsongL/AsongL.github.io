Java基础 - final

final是什么？

final是Java语言中的一个关键字

表示最终的，不可变的

可以修饰变量以及方法，还有类等

final修饰的表示最后的，不能变的，不能改的

final修饰的变量？

final修饰的变量只能赋值一次

final修饰的方式？

调用子类中特有的方法时，需要向下转型

final修饰的方法不能被覆盖

final修饰的类？

final class A{} // A没有子孙

final修饰的类无法继承

final修饰的变量，如果这个变量是一个“引用”会怎样？

final修饰的引用只能指向1个对象，并且它只能永远指向该对象，无法再指向其他对象
并且再该方法执行过程中，该引用指向该对象之后，该对象不会被垃圾回收器回收 直到方法结束 才会释放空间
final修饰的变量只能赋值一次
“引用”是不是一个变量？是

eg: final Person p = new Person(30);

p = new Person(30); // 报错 因为p已经被赋值一次

引用指向的对象不能换，但是对象的数据可以变
虽然final的引用指向对象A后，不能再重新指向对象B，但是B中的数据可以被改变

final修饰的实例变量呢？

实例变量如果没有手工赋值的话，系统会赋默认值
实例变量加final的话 必须手动赋初值

eg:final int age = 0;
实例变量
构造方法执行的过程中赋值（new的时候)
只要我赶在系统赋默认值之前赋值就行
final实例变量系统不负责赋默认值
这个赋值在变量后面赋值可以，在构造方法中赋值可以

final修饰的实例变量一般添加static修饰
该实例变量前面应该添加，static，变成静态变量节约空间

常量和静态变量的区别？

static final 常量 全部大写中间
常量和静态变量，都是存储在方法区的，并且都是在类加载时初始化
常量一般都是公共的 public的
